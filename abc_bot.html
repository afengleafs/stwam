ABC 栈概览

Scalable Behavior Cloning with Open Data, Training, and Evaluation

A. ABC-130K 数据集

官网口径约为 134,806 episodes、195 个任务、3,553 小时双臂操作数据，覆盖 pick-and-place、folding、 handover、insertion、tool use、assembly 等操作原语。Hugging Face 数据页存在略不同统计口径， 但量级一致。

硬件是双臂 YAM station：2 个 6-DoF YAM 机械臂 + 平行夹爪；视觉输入包括顶部相机和两个腕部相机。

B. ABC-DiT

ABC-DiT 是扩散/flow 风格策略，使用较小视觉 backbone 和较大 action head。 Table 2 中 ABC-DiT 为 85.7M backbone + 1.93B action head。

C. ABC-VLA

ABC-VLA 使用 Gemma 3 VLM backbone 和轻量 DiT action head。核心思想是把大 VLM 当条件编码器， 动作生成交给小型 diffusion/flow head，而不是把动作完全离散成 token 自回归生成。

D/E. ABC-Sim 与 ABC-Eval

ABC 还发布 MuJoCo 仿真和约 400 小时 sim-teleop 数据。论文报告 sim strict success 与 real strict success 相关系数为 r=0.85，sim progress 与 real progress 为 r=0.91。

真实评测侧，training loss 和 validation action error 与真实表现更相关；普通 validation loss 相关性弱。

训练与推理效率

小 action head 是 ABC-VLA 的核心系统收益

k=8 multiple diffusion draws

对 AdaLN 和普通 X-Attn，扩散损失同时对 data sample、noise、timestep 取期望。 因为 ABC-VLA 的 DiT head 只有约 42M/44.7M，而 VLM 有 4B+，可以一次 VLM forward 后复制 k 份 hidden state 和 clean action，配 k 个独立 noise/timestep。

论文报告 k=1 到 k=8 只从 1.346 s/step 增到 1.366 s/step（H200），但降低了梯度估计方差。

推理速度反直觉

ABC-DiT 是 85.7M backbone + 1.93B action head；ABC-VLA 是 4.3B backbone + 44.7M action head。 ABC-VLA 总参数更多，却在 RTX 5090、10 diffusion steps、fullgraph compile 后达到 17.5 ms / 57.2 Hz。

原因是 ABC-VLA 的 VLM 只跑一次并缓存 features，10 步去噪只重复 45M 小头；ABC-DiT 每步都要重复跑约 1.93B 的大 DiT head。

模型 Backbone Action head 10-step latency Hz 重复计算瓶颈
ABC-DiT 85.7M 1.93B 36.3 ms 27.6 每个 denoising step 重跑大 DiT action head。
ABC-VLA 4.3B 44.7M 17.5 ms 57.2 VLM 只跑一次；denoising 重复部分是小 DiT head。

RTC 共同点

部署侧两者都与 Real-Time Chunking 思路兼容。ABC 的策略训练使用 training-time RTC，并在执行时用 action prefix 避免 chunk 间 mode switching、提升轨迹平滑性。Appendix H.2 也提醒：更长 prefix 更平滑，但可能降低视觉响应性。

写进 Tac-π0.5 的可用表述

Related work / 对比表 / ablation discussion
  • 为“轻量 action head + 大 backbone 单次前向”辩护：ABC-VLA 用 4.3B VLM + 44.7M DiT head， 通过 8 个 query pooled VLM features 调制小 DiT，在 7,000 小时内部数据、200k steps、三项真实机器人任务上达到 61.4% mean progress。
  • 讨论 π0.5-style 深耦合 action expert 的风险：ABC 比较了 layer-wise VLM KV reuse 风格接口，发现它不如最简单的 final-layer X-Attn， 而 final-layer X-Attn 又远低于 pooled adaLN。
  • 讨论训练效率：ABC-VLA 的 k=8 diffusion draws 共享大 VLM 前向/反向，只重复小 head，把扩散目标方差降低和吞吐结合起来。
  • 讨论系统定位：π0.5 证明数据混合和层级语义推理的开放世界泛化价值；ABC-VLA 提供更清晰的 action-head connector 消融，两者互补，不应简单写成替代关系。
